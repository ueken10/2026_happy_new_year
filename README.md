# 2026 HAPPY NEW YEAR - Flying Logo

Three.jsとTone.jsを使った新年のフライングロゴアニメーション

## 機能

- **3Dテキスト**: TextGeometryによる"2026 HAPPY NEW YEAR"の立体テキスト
- **アニメーション**: 15秒間のループアニメーション
  - **0-12秒**: ダイナミックな移動フェーズ
    - 奥（z=-100）から手前（z=10）への移動
    - X、Y、Z軸の回転アニメーション
    - スケール変化（0.5→1.0）
    - 虹色カラーグラデーション
  - **12-15秒**: 静止フェーズ
    - テキストが正面（回転なし）で静止
    - オーケストラヒット風の和音再生
- **エフェクト**:
  - パーティクル背景（1000個の星）
  - 動的なポイントライト（2個）
  - フォグ効果
  - ワイヤーフレームの地面
- **音楽**: Tone.jsによるオーケストラヒット（12秒時点で自動再生）

## 使い方

1. `index.html`をブラウザで開く
2. ページ読み込み時に音声が自動で初期化されます
3. 15秒のアニメーションを楽しむ（自動でループします）
   - 12秒時点でオーケストラヒットが鳴ります

## アニメーション詳細

### タイムライン
- **0-12秒**: 移動・回転・カラー変化
  - イージング関数: `1 - (1 - t)³`
  - 回転: Y軸で4回転、X軸とZ軸で正弦波
- **12秒**: オーケストラヒット（C3, E3, G3, C4, E4, G4の和音）
- **12-15秒**: 正面向きで静止
- **15秒**: ループ開始（0秒に戻る）

### ビジュアル要素
- **カメラ**: 視野角75°、位置z=30
- **背景**: グラデーション（#0a0a2e → #16213e → #0f3460）
- **フォグ**: 深度10-100
- **ライティング**: 
  - アンビエントライト（明るさ2）
  - ディレクショナルライト（シャドウあり）
  - 2個の動的ポイントライト（ピンク・シアン）
